# Canva Clone Practice Project

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase (for backend)
- Vercel (for deployment)

## Getting Started
1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Open `http://localhost:3000` in your browser to view the app.
